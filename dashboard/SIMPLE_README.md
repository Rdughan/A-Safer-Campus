## ğŸš¨ Troubleshooting

### **"Connection Failed" Error**
- Double-check your Supabase URL and key
- Make sure your Supabase project is active
- Ensure you're using the **anon public key**, not the service role key

### **"No Data" Showing**
- Check that your `incidents` table has data
- Verify the table name is exactly `incidents`
- Make sure your RLS policies allow public read access

### **Charts Not Loading**
- Check your internet connection
- Try refreshing the page
- Make sure Chart.js is loading (check browser console)
